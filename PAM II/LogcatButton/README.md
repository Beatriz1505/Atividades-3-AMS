# Logcat Button - My Chemical Romance Edition

Aplicativo Android desenvolvido com foco em uso de Logcat, utilizando uma interface inspirada na estética da banda My Chemical Romance.

---

# Sobre o projeto

O aplicativo permite que o usuário insira seu nome e, ao clicar nos botões, diferentes mensagens são registradas no Logcat com níveis variados (Error, Warning, Debug e Info).

Cada botão representa um álbum da banda.
